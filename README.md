# Hybrid Cloud Security System

A Django-based web application that demonstrates secure file handling and access control in a hybrid cloud environment.

---

## Features

- User authentication
- Secure file upload
- Environment variable based secret management
- Clean and secure GitHub repository

---

## Tech Stack

- Python
- Django
- SQLite
- Git & GitHub

---

## Project Structure

hybrid-cloud-security/
- hybrid_cloud_security/
- securityapp/
- manage.py
- .gitignore

---

## Security Notes

- Secret keys are not stored in the repository
- Sensitive files are ignored using .gitignore
- Environment variables are used for secrets

---

## How to Run

1. Clone the repository
2. Install dependencies
3. Run migrations
4. Start the server
